dascore.io.core module¶

Base functionality for reading, writing, determining file formats, and scanning Das Data.

class dascore.io.core.FiberIO[source]¶

Bases: ABC

An interface which adds support for a given filer format.

This class should be subclassed when adding support for new Patch/Spool formats.
