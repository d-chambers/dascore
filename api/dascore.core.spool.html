abstract chunk(overlap=None, keep_partial=False, snap_coords=True, tolerance=1.5, **kwargs)[source]¶

Chunk the data in the spool along specified dimensions.

Parameters:

  • overlap (Union[int, float, str, datetime64, Timestamp, None]) – The amount of overlap between each segment, starting with the end of first patch. Negative values can be used to induce gaps.

  • keep_partial (bool) – If True, keep the segments which are smaller than chunk size. This often occurs because of data gaps or at end of chunks.

  • snap_coords (bool) – If True, snap the coords on joined patches such that the spacing remains constant.

  • tolerance (float) – The number of samples a block of data can be spaced and still be considered contiguous.

  • kwargs – kwargs are used to specify the dimension along which to chunk, eg: time=10 chunks along the time axis in 10 second increments.
